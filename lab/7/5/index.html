IST 363 - Advanced Web Development

Lab 7 step 5

resetGame and gameOver Functions

The resetGame function will reset players' scores and text from index.html

  • Set playerScore and computerScore equal to 0
  • Set the innerHTML of playerScoreText and computerScoreText to nothing

The gameOver function will alert the user of who won the game and then call the resetGame function so a new game can be started

  • Use an if/else conditional to create an alert telling the player if they won or if they lost
  • After alerting the player, invoke the resetGame function
